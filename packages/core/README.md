# 慕课网前端统一研发脚手架

## About

慕课网前端架构师课程专属脚手架

## Getting Started

安装：

```bash
npm install -g @freelog-cli/core
# OR
yarn global add @freelog-cli/core
```

创建项目

```bash
freelog-cli init 
```

发布项目

```bash
freelog-cli publish
```

## More

清空缓存：

```bash
freelog-cli clean
```

DEBUG 模式：

```bash
freelog-cli --debug
```

指定本地包：

```bash
freelog-cli init --packagePath /Users/sam/Desktop/freelog-cli/packages/init/
```
